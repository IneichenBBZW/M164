Schritt 1
Datenbank und Tabellen erstellen.
Es gibt dafür ein fertiges Script.
-----------------------------------------------------------
-- Datenbank erstellen
-----------------------------------------------------------
use master
go
drop database if exists FahrzeugPark;
go
create database FahrzeugPark;
go
use FahrzeugPark;
go

-----------------------------------------------------------
-- Tabellen erstellen
-----------------------------------------------------------
create table Mitarbeiter (
  MitarbeiterId integer identity,
  Nachname nvarchar(30),
  Vorname nvarchar(30),
  primary key(MitarbeiterId)
);

create table Fahrzeug (
  FahrzeugId integer identity,
  Marke nvarchar(50),
  Typ nvarchar(50),
  NeuPreis decimal(9, 2),
  Jahrgang integer,
  Occasion bit,
  fk_MitarbeiterId integer,
  primary key(FahrzeugId),
  foreign key(fk_MitarbeiterId) references Mitarbeiter(MitarbeiterId)
);

Schritt 9
ZeitWert anpassen.
Alter unter 6 → 50%
Alter 6 bis 25 → 10%
Alter 26 bis 50 → 300%
Alter 51 bis 75 → 1000%
Alter 76 bis 100 → 10000%
Alter über 100 → 100000%
Tipp: aktuelles Jahr mit year(getdate()).

Schritt 10
Occasionen.
ZeitWert minus 20%.
Nur wenn ZeitWert über 100000 ist.
